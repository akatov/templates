## Repository Purpose

This is a development templates repository containing reusable project templates.

## Development Environment

This project uses a devcontainer-based development setup:

- **Runtime**: Node.js 24 (TypeScript)
- **Package Manager**: pnpm
- **Formatter**: dprint
- **Linting**: ESLint
- **Testing**: Vitest

## Setup

Copy `.devcontainer/devcontainer.env.example` to `.devcontainer/devcontainer.env` before starting the devcontainer.
